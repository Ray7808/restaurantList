# 餐廳清單

此專案主要用來串接餐廳 API 並顯示、搜尋餐廳名稱及詳細資訊。

## 產品功能

-   串接餐廳 API 並搭配模板引擎(handlebars)，顯示餐廳資訊。
-   點進餐廳圖片，可以顯示該餐廳的完整資訊。
-   製作搜尋欄，可以輸入並查找想找的餐廳名稱或是種類。

## 安裝流程

1. git clone 下載檔案
    ```
    git clone https://github.com/Ray7808/restaurantList.git
    ```
2. 安裝相關套件

    ```
    npm install
    ```

3. 執行程式
    ```
    npm run start
    ```

## 使用的套件

-   Bootstrap
-   Express
-   Handlebars
-   Nodemon
